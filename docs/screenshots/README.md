# Screenshots

Drop the following image files here so the main [README](../../README.md)
renders them. Filenames must match exactly:

| File | What it shows |
|------|----------------|
| `live.png` | The live view, clean/fullscreen (the garden shot) |
| `interface.png` | The live view with the settings tray open and the motion panel visible |
| `stats.png` | The Statistics page (temp chart, activity, animals card) |

PNG or JPG both work; if you use `.jpg`, update the extensions in `README.md`.
